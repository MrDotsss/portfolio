💎 JBB Gold – Jewelry Shopping Application

Full-stack e-commerce system with 3D viewer and AR virtual try-on.

JBB Gold is our thesis project—a complete jewelry shopping platform for both web and mobile.
This was my first experience as a full-stack developer, where I explored both web and mobile application development, adapting quickly and working across multiple technologies.

View Sample Site Download Sample App

🖥️ Web Admin Panel (Desktop – Admin Use)

  • Vue.js + Pinia: Built a dynamic frontend with modular state management for managing products and orders.
  • Node.js + Express: Developed RESTful APIs for backend logic and secure data handling.
  • Firebase Authentication: Implemented secure user authentication for admin access.
  • MongoDB: Designed and managed scalable NoSQL database schemas.

This experience taught me how to structure scalable codebases, manage state efficiently, and connect frontend and backend seamlessly.

📱 Mobile App (Client Use – Android)

  • Flutter + Riverpod: Built a responsive Android app with structured state management.
  • Snap AR CameraKit: Enabled virtual try-on using augmented reality technology.
  • Google Model Viewer: Embedded interactive 3D jewelry previews.
  • PayMongo: Integrated real-time payment processing within the app.

This part of the project enhanced my understanding of mobile UX, third-party SDK integration, and mobile-first design practices.
